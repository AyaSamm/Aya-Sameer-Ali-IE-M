What are the four tags required in every HTML page?

   1.The "! DOCTYPE html" declaration defines this document to be HTML5 and helps browsers 
   to display web pages correctly. It must only appear once, at the top of the page (before any 
   HTML tags).
   2.The "html" element is the root element of an HTML page and it is end with tag "/html"
   3.The "head" element contains meta information about the document, it is end with tag "/head"
   4.The "body" element contains the visible page content, end tag is "/body"
